shuffle_test(measure, x, y, z=False, ns=200, ci=0.95, **kwargs)

source code 

Shuffle test Repeatedly shuffle the x-values and then estimate measure(x,y,[z]). Returns the mean and conf. interval ('ci=0.95' default) over 'ns' runs. 'measure' could me mi,cmi, e.g. Keyword arguments can be passed. Mutual information and CMI should have a mean near zero.
